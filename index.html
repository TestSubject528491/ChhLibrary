Util.bound(x, lower, upper)
Description
Returns the number closest to x within the closed interval [lower, upper].
Param x
the number to be tested
Param lower
the minimum of the interval
Param upper
the maximum of the interval (if < lower, NaN is returned)
Util.isPrime(n)
Description
Determines whether n is prime.
Param n
a positive integer to be tested
